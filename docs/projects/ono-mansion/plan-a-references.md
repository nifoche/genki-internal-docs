## 7. 日本の防音等級規格

### 参考情報
- **JIS A規格**（遮音等級）:
  - JIS A-1: 35dB以上（最高級）
  - JIS A-2: 30dB以上
  - JIS A-3: 25dB以上
  - JIS A-4: 20dB以上
  - JIS A-5: 15dB以上

- **LL規格**（床衝撃音）:
  - LL-40: 最高級（マンション推奨）
  - LL-45: 高性能（集合住宅向け）
  - LL-50: 標準（一般住宅向け）
  - LL-55: 基礎
  - LL-60: 最低限
