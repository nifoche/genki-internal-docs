## 計算方法

### 地域別価格調整

愛知県の地域調整率: **1.035**（全国平均1.00に対して+3.5%）

```python
def calculate_regional_adjustment(base_price: float, region: str = "aichi") -> float:
    """
    地域別価格調整率を適用

    参考データ:
    - 愛知県: 1.02 - 1.05（名古屋市エリア）
    - 全国平均: 1.00
    - 東京: 1.05 - 1.10
    - 大阪: 1.03 - 1.08
    """
    regional_factors = {
        "national": 1.00,
        "aichi": 1.035,   # 愛知県の平均（1.02-1.05の中央値）
        "tokyo": 1.075,   # 東京の平均
        "osaka": 1.055,   # 大阪の平均
    }

    factor = regional_factors.get(region, 1.00)
    adjusted_price = base_price * factor

    # 切り上げ（リフォーム業界の慣習）
    return math.ceil(adjusted_price)
```

### 丸め処理

1. **単価計算後**: 切り上げ（`math.ceil`）
2. **総額算出後**: 百円単位で丸め込み

# 百円単位で丸め込み
