## 適用シナリオ

### プロ推奨コースが推奨されるケース

✅ **適している**
- オフィスでのWeb会議・通話が多い
- ボーカル録音・ナレーション録音を行う
- プレゼンテーション・重要な会議を行う
- 長期的な安定性を重視する
- 3年保証を希望する

❌ **適していない**
- 予算¥40万未満に抑えたい（→ プロ標準コース）
- 楽器演奏を行う（→ プロプレミアムコース）
- 転居の可能性が高い（→ DIYコース）

## 注意事項

### 別途発生する可能性のある費用

| 項目 | 費用 | 条件 |
|------|------|------|
| **現場調査費** | ¥10,000〜20,000 | 施工前に必須 |
| **既存撤去費用** | ¥20,000〜50,000 | 既存の内装材・設備撤去が必要な場合 |
| **下地補修費用** | ¥10,000〜30,000 | 既存壁・床の損傷が大きい場合 |
| **電気工事追加** | ¥30,000〜80,000 | コンセント増設・配線引き直し等 |
| **管理会社承認料** | ¥20,000〜50,000 | 賃貸物件の管理会社手数料 |
| **原状回復費用** | ¥50,000〜150,000 | 転居時の完全復旧 |

### 賃貸物件特有の注意点

1. **原状回復可能性の事前確認**
   - 施工前に管理会社に確認書取得
   - 「浮き工法」は原則として可能
   - 畳敷設は管理会社次第

2. **騒音規制の遵守**
   - 近隣住民への事前説明推奨
   - 施工時間帯は8:00〜18:00に遵守
   - 振動ドリル等の使用は事前相談

3. **共用部分の使用**
   - ダクト配線に共用部分を使用する可能性
   - 管理組合の承認が必要
