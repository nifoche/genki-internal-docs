## プロジェクト概要

|------|------|

| D1動画の目標 | バズによるフォロワー獲得・シリーズ視聴習慣の形成 |

| 顔出し | なし |

## アルゴリズム設計の前提（2026年最新）

ClaudeCodeはこの数値を全シーンの設計基準にすること。

| KPI | 目標値 | 意味 |
|-----|--------|------|
| 3秒視聴維持率 | **60%以上** | この数値を超えるとバズ確定ルートへ |
| 保存率 | **1.5%以上** | 「見返す価値」がある設計が必要 |
| シェア率 | **0.5%以上** | 発見タブへの到達条件 |
| 視聴完了率 | **50%以上** | 平均再生時間で測定 |

**Instagramアルゴリズムが評価する順番：**
1. 再シェア
2. 最後まで視聴
3. いいね
4. 音源ページへの遷移

## D1動画｜シーン構成（45秒）

### Scene 0｜ループ誘発ショット（0〜1秒）
```
内容   : 完成後の猫部屋イメージ（AI生成画像）が0.5秒映り、瞬時に「DAY 1」へ切替
目的   : 最後のシーンと同カットにしてループ再生を誘発
音     : 無音
演出   : フラッシュカット
```

### Scene 1｜冒頭メッセージ「猫目線-3 × C-2合体」（1〜15秒）

**[1〜6秒] 猫からの挨拶（感情で掴む）**
```
画面   : 真っ黒
テキスト（順番に表示）:
  「はじめまして。」
  （1秒の沈黙）
  「まだ会ったことないけど、」
  「ここに引っ越してくる予定の猫です。」

SE     : 猫の足音（ぺたぺた）→ にゃっ
ナレーション（四国めたん）:
  「はじめまして。
   まだ会ったことないけど、
   ここに引っ越してくる予定の猫です。」
```

**[6〜15秒] 数字で現実を叩きつける（事実で正当化）**
```
画面   : 黒背景に数字が順番にカウントアップ→ダウン
テキスト（タイミングをずらして出現）:
  「日本の賃貸、約2,600万戸。」  → 2秒表示
  「猫可、約100万戸。」          → 2秒表示
  「猫専用設計、0戸。」          → 1秒表示・「0」だけ赤・震えるエフェクト
  （0.8秒の沈黙）
  「今日から、1。」              → 大きく・白・中央

ナレーション:
  「日本の賃貸、約2600万戸。
   猫可、約100万戸。
   猫専用設計、0戸。
   …今日から、1。」
```

### Scene 2｜プロジェクト宣言（15〜28秒）

```
画面   : 進捗バーUI + DAYカウンター + 設備ロックリスト

表示要素:
  ┌─────────────────────────┐
  │  DAY 1 / 39日後 完成予定   │
  │  ▓░░░░░░░░░░░░░░░░░░  1/39│
  ├─────────────────────────┤
  │  🔒 キャットウォーク        │
  │  🔒 猫専用ペットドア        │
  │  🔒 爪とぎ対応壁クロス      │
  │  🔒 消臭・抗菌クロス        │
  │  🔒 ??? （隠し設備）        │
  └─────────────────────────┘
  「フォロワーが増えるたびに、設備が解放されます。」

ナレーション:
  「39日後に完成するこの部屋。
   何人集まるかで、仕様が変わります。
   今日から、全部見せます。」
```

### Scene 3｜視聴者参加（コメント誘発）（28〜38秒）

```
画面   : 投票UI風テロップ（アニメーションで順次表示）

テキスト:
  「あなたが猫なら、どこが欲しい？」

  A 🐾 キャットウォーク（天井まで）
  B 🪟 猫専用の窓
  C 🧱 爪とぎ放題の壁
  D 💤 日当たり最高の昼寝スポット

  「コメントで教えて。本当に作ります。」

ナレーション:
  「コメントに書いてくれたもの、
   できる限り作ります。約束します。」
```

### Scene 4｜CTA + ループ仕掛け（38〜45秒）

```
画面   : カウントダウン数字がカチカチ回る
  39 → 38 → 37 → ... → 1 → 完成

テキスト:
  「完成まで、一緒にいてください。」

最後0.5秒 : Scene 0と同じ完成イメージカット（ループ誘発）

ナレーション:
  「39日後、答え合わせをしましょう。」
```

## ClaudeCodeへの実装指示

```bash
npm install
```

