### ポトス（最も一般的な有毒植物）

**特徴**:

## 第4章 LED育成ライトの適正配置

### 配置ガイドライン

**PPFDと距離の関係**:
- PPFD 100-200: 距離 30-50cm（小型植物）
- PPFD 200-300: 距離 20-40cm（中型植物）
- PPFD 300-500: 距離 30-60cm（大型植物）

**配置例（401号室）**:

```
┌─────────────────────────────┐
│  窓                          │
│  ┌─────┐  ┌─────┐  ┌─────┐   │
│  │LED  │  │LED  │  │LED  │   │
│  └─────┘  └─────┘  └─────┘   │
│     ↓        ↓        ↓      │
│  [植物1]  [植物2]  [植物3]    │
│  (小型)   (中型)   (大型)    │
└─────────────────────────────┘
```

**注意点**:
- LED近づけすぎ（<30cm）: 葉焼けのリスク
- LED遠すぎ（>60cm）: 光量不足
- タイマー使用: 12-14時間/日（朝7時〜夜7時）

---

### 点灯時間の最適化

**季節別点灯時間**:
- 夏期: 12時間点灯（朝7時〜夜7時）
- 冬期: 14時間点灯（朝7時〜夜9時）
- 中間期: 13時間点灯（朝7時〜夜8時）

**タイマー設定**:
- 朝7時: 点灯
- 夜7時（夏期）・夜9時（冬期）: 消灯
- 24時間タイマー使用

---

### 水やりのヒント

**水やりの頻度**:
- 夏期: 週1回
- 冬期: 2週間に1回
- 中間期: 10日に1回

**水やりの時間帯**:
- 朝7時〜9時（推奨）
- 夕方5時〜7時（可）
- 避ける: 正午（暑すぎる）

**水やりの方法**:
1. 土が乾いているか確認（指で触る）
2. 鉢底から水が流れ出るまで水やり
3. 受皿に溜まった水を捨てる（根腐れ防止）
4. 葉に水がかからないように注意

---

### 湿度管理

**適正湿度**:
- 観葉植物: 50-70%
- 人間の快適湿度: 40-60%
- 猫の快適湿度: 50-60%

**湿度低下時の対策**:
- 加湿器導入（¥10,000〜¥20,000）
- 観葉植物をまとめる（蒸散効果で湿度上昇）
- トレイに水を張る（水の蒸発で湿度上昇）

**湿度上昇時の対策**:
- 換気（24時間換気システム活用）
- エアコン（除湿機能）
- 風通しを良くする
