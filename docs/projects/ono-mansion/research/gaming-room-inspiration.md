## 部屋レイアウト

### 最小スペース: 6畳〜8畳

**基本レイアウト**:
```
┌─────────────────────────────────┐
│                                 │
│  [防音ブース]    [ゲーミングデスク]  │
│   1.5〜2畳      120cm以上         │
│                                 │
│         [モニター×3]            │
│                                 │
│      [オフィスチェア]            │
│                                 │
└─────────────────────────────────┘
```

**ポイント**:
- デスクは窓側に（自然光を活用）
- 防音ブースは部屋の奥に配置
- ケーブル配線を隠す（壁際にまとめる）

### 小スペースでの工夫

**3.5m × 2.5mの実例**:
- ベッドを二段構えにしてデスクスペース確保
- 壁面収納を活用
- モニターアームでデスク上をスッキリ

## 防音対策

### 賃貸でも可能な防音

**レベル1: 軽度対策（予算1〜3万円）**
- 防音カーテン
- 防音マット・カーペット
- 隙間テープ
- 本棚・クローゼットを活用した吸音

**レベル2: 中度対策（予算5〜10万円）**
- デスク用防音ブース
- 壁に吸音パネル（賃貸OKの粘着式）
- 防音パーテーション

**レベル3: 本格対策（予算30〜50万円）**
- ユニット式防音室（GAMEBOX等）
- 完全防音ブース（1.5〜2畳）
- 専用換気システム

### ユニット式防音室

**GAMEBOX**（テックセル）:
- 1-2畳の完全防音室
- 数時間で組立・解体可能
- 賃貸物件でも設置OK
- 引越し時も持ち運び可能

**TOKYO SOUND BOX**:
- VTuber・ライバー向けセットアップ
- 現地調査・音响調整込み
- 賃貸対応の防音リフォーム

**参考URL**:
- [GAMEBOX公式サイト](https://teccell.co.jp/gamebox-lp/)
- [TOKYO SOUND BOX](https://www.tokyosoundbox-lab.com/forlivers)
- [自宅用簡易防音室・防音ブースおすすめ8選](https://create.adrise.jp/column/workbooth/341/)
- [VTuber配信防音対策](https://www.soundzone.jp/staffblog/13214/)

## 配信機材

### 必要機材一式

**PC**:
- CPU: Intel Core i7/i9 または AMD Ryzen 7/9
- GPU: RTX 4070以上（4K配信対応）
- RAM: 32GB以上
- ストレージ: 1TB NVMe SSD

**モニター**:
- メイン: 27インチ 4K 144Hz
- サブ: 27インチ QHD 144Hz×2
- モニターアーム必須

**カメラ**:
- 4Kウェブカメラ（Sony ZV-E10等）
- リングライト

**マイク**:
- コンデンサーマイク（Shure SM7B等）
- オーディオインターフェース
- ポップガード

**参考URL**:
- [Dezctop - Gaming Room Setup Ideas](https://dezctop.com/blog/setup-inspirations/gaming-room-setup)
- [BenQ - Gaming Room Ideas](https://www.benq.eu/en-cee/knowledge-center/knowledge/what-should-my-gaming-room-must-have.html)

## ライティング

### RGBライティング

**効果**:
- 配信画面の映えを向上
- 部屋の雰囲気を作る
- カメラ映えを意識

**配置**:
- デスク背面: LEDテープ（RGB対応）
- モニター背面:_bias lighting_（目の疲れ軽減）
- 天井: シーリングライト（調光可能）

## 快適性

### 長時間座れる椅子

**要件**:
- ランバーサポート（腰サポート）
- 通気性メッシュ素材
- 4Dアームレスト
- リクライニング機能

**おすすめ**:
- Herman Miller Aeron
- Steelcase Leap
- 日本製オフィスチェア（オカムラ等）

### 換気・温度管理

**必需品**:
- 空気清浄機（PCの熱対策）
- サーキュレーター
- エアコン（室温24〜26度維持）

### 配線管理

**ポイント**:
- ケーブルトレイでデスク下をスッキリ
- タップ周りをまとめる
- 無線化できるものは無線化（キーボード・マウス）

**参考URL**:
- [List-Em - Streaming Gaming Room Ideas](https://www.list-em.com/top-ten-streaming-gaming-room-ideas/)
