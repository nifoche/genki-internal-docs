> 作成日: 2026-02-04
> コンセプト: D: ボタニカル特化
> 実験期間: 2週間（Week 5-6）
> 投資額: 3.4万円
> 成功基準: 問い合わせ5件以上・内見2件以上・成約1件以上

---

## 実験目的

1. **市場反応の検証**: 観葉植物・インテア関心層の需要を確認
2. **賃料設定の妥当性**: 4.3〜4.5万円で成約可能か検証
3. **差別化の効果**: SNS映えコンセプトの集客力を測定
4. **回収期間の実証**: 7ヶ月での回収が現実的か検証
5. **スケールアップの可否**: 402号室・他物件への展開判断

## デイ13-14: 設置施工（5時間）

### 施工前準備

- [ ] 部屋の掃除（床・窓・クローゼット）
- [ ] レイアウトの決定（植物・ライト・棚の位置）
- [ ] コンセント位置の確認
- [ ] 工具の準備（ドリル・水平器・メジャー）

### 施工手順

**1. 壁穴あけ（2時間）**:
- [ ] 簡易棚の位置決め
  - 高さ: 視線よりやや上（約160cm）
  - 間隔: 2箇所（対面壁に1箇所ずつ）

- [ ] 下書き（鉛筆で位置をマーク）
- [ ] 水平器で水平確認
- [ ] ドリルで穴あけ（アンカー埋め込み）
- [ ] 簡易棚の取り付け
- [ ] 耐荷重確認（植物を置いても問題ないか）

**2. LEDライト設置（1時間）**:
- [ ] LED育成ライトの設置
  - 場所: デスク横・窓際
  - 高さ: 植物の上から照らす位置
  - 固定: スタンドまたは壁掛け

- [ ] LEDテープライトの設置
  - 場所: 壁の上部（天井近く）
  - 貼り付け: 両面テープで貼付
  - 配線: 延長コードでコンセントへ

- [ ] 点灯確認
- [ ] 照明の角度・明るさ調整

**3. 観葉植物配置（1時間）**:
- [ ] ポトスをハンギング（天井から吊る下げ）
- [ ] ドラセナをコーナーに配置
- [ ] サンスベリアをデスク横に配置
- [ ] ユッカを窓際に配置
- [ ] 多肉植物を簡易棚に配置
- [ ] 植木鉢のデザイン統一

**4. 最終調整（1時間）**:
- [ ] 全体のバランス確認
- [ ] LEDライトの点灯パターン調整
- [ ] 植物の葉の拭き掃除
- [ ] 床の掃除（土・葉落ちの処理）

### 施工後チェック

- [ ] 全てのライトが点灯するか
- [ ] 植物が安定しているか
- [ ] コンセントに過負荷がないか
- [ ] 延長コードがつまらないか
- [ ] 写真撮影前の最終確認

### 撮影テクニック

**日当たり**:
- 曇りの日または午前中に撮影（光が柔らかい）
- 直射日光を避ける（カーテンを引いて）
- 自然光をメインに、LEDライトを補助的に使用

**構図**:
- 三分割法（画面を縦横3等分して、交点に被写体を配置）
- 視線を意識（低い位置から撮って広さを出す）
- 植物を前景に配置（奥行きを出す）

**露出**:
- 明るさを+0.3〜+0.7に補正（明るく鮮やかに）
- コントラストを少し下げる（柔らかい印象）
- 彩度を少し上げる（植物の締を鮮やかに）

## トラブルシューティング

### 問い合わせが来ない場合

**対策**:
1. 写真の撮り直し（明るさ・構図の改善）
2. 掲載内容の改善（キャッチコピーの変更）
3. SNS投稿頻度の増加（週3回→毎日）
4. 有料広告の検討（Instagram広告・SUUMO有料掲載）

### 内見からの成約がない場合

**対策**:
1. 賃料の引き下げ（4.5万円→4.3万円）
2. 初期費用の割引（敷金無料キャンペーン）
3. 特典の追加（1ヶ月無料、Amazonギフト券プレゼント）
4. 内見スキルの向上（説明の改善・デモ時間の短縮）

### 植物が枯れる場合

**対策**:
1. 水やりの頻度調整
2. 日当たりの確保（カーテン開放）
3. 肥料の追加
4. 枯れた植物の交換
