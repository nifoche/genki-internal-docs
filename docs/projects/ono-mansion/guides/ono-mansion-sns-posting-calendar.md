## トラブルシューティング

### 問題1: 反響が少ない場合

**対策**:
- [ ] 投稿時間を変えてみる（朝8時→朝7時など）
- [ ] ハッシュタグを増やす（10個→20個）
- [ ] 画像を変更する（より鮮明な写真を使用）
- [ ] ストーリーズでリマインダー
- [ ] 友人・知人に拡散依頼

### 問題2: スパムコメントがつく場合

**対策**:
- [ ] コメント制限をオンにする（フォロワーのみ）
- [ ] 不適切なコメントを非表示にする
- [ ] 迷惑行為にはブロック対応

### 問題3: DMに返信がない場合

**対策**:
- [ ] 1日待ってからフォローアップ
- [ ] 別のプラットフォームで連絡（Twitter→Instagramなど）
- [ ] 返信を催促しない（相手のペースに任せる）

### 問題4: 問い合わせが想定より少ない場合

**対策**:
- [ ] SUUMO・HOMESの掲載内容を見直し
- [ ] 賃料を下げる（4.3万円→4.2万円）
- [ ] 初月キャンペーン（初月50%OFF）
- [ ] SNS広告の出稿（Instagram広告5,000円分）
