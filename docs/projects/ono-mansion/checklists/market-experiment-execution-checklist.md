# 小野マンション・市場実験実行チェックリスト

> 作成日: 2026-02-05
> 対象: プランC（ハイブリッド）Phase 1
> 目的: 401号室・低投資ボタニカルの市場実験を実行
> 期間: Week 1-4（1ヶ月間）

---

## チェックリスト概要

**使用方法**:
- 各タスクの `[ ]` を完了時に `[x]` に書き換えてください
- メモ欄に実施日・担当者・気づきを記録してください
- 週末に進捗を確認し、次週の計画を調整してください

**進捗管理**:
- 週次レビュー: 毎週日曜日に進捗確認
- 遅延タスク: 週次レビューで再計画
- 成功基準: 問い合わせ3件以上・内見1件以上・成約1件以上

## トラブルシューティング（Phase 2）

### ペットの臭いリスク

**症状**:
- 部屋に猫の臭いが充満する
- 近隣から苦情が来る

**原因**:
- 換気不足
- 猫トイレの掃除不足
- 空気清浄機のフィルター目詰まり

**対策**:
- [ ] 24時間換気システムの強化（風量UP）
- [ ] 空気清浄機のフィルター交換（時期短縮）
- [ ] 消臭剤追加（ペット用消臭剤）
- [ ] 猫トイレの清掃頻度UP（1日2回〜3回）


### 空室リスク

**症状**:
- 問い合わせが来ない
- 内見キャンセルが多い

**原因**:
- ニッチ市場（猫多頭飼い）のため、入居者決定まで時間がかかる
- 賃料が高い（6.15万円）

**対策**:
- [ ] SUUMO・HOMESの掲載強化（写真追加・文言修正）
- [ ] SNS広告の活用（Instagram広告5,000円）
- [ ] 紹介制度（紹介で1ヶ月無料）
- [ ] ペットショップとの提携（入居者紹介）
- [ ] 賃料下げ検討（6万円〜5.8万円）


### クッションフロア・壁パネルの破損リスク

**症状**:
- クッションフロアに穴が開く
- 壁パネルが剥がれる

**原因**:
- 猫の引掻き過多
- 耐久性の低い素材

**対策**:
- [ ] 爪とぎ器設置（麻縄・ダンボール等）
- [ ] キャットタワー設置（運動不足解消）
- [ ] 補修用品の確保（予備のクッションフロア・壁パネル）
- [ ] 入居者へのマニュアル提供（猫のしつけ）
