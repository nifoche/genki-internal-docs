## 📋 チェックリスト使用方法

### 確認項目の見方

| 項目 | 説明 |
|------|------|
| **[ ]** | 未実施項目 |
| **[x]** | 実施済み項目 |
| **確認方法** | 目視・テスト・測定など具体的な確認手法 |
| **合格基準** | 具体的な数値や状態（必ず守るべき最低基準） |
| **写真** | 必須撮影箇所（Before/After記録用） |
| **担当署名** | 実施者と確認者の両署名 |

### 品質管理の原則

1. **問題即修正**: 不合格項目があれば次に進まず、即座に修正
2. **写真記録**: 全ステップの撮影で施工品質を証明
3. **署名確認**: 施工担当者と所有者の双方署名で完了を確認
4. **原状回復優先**: 賃貸物件であることを意識し、復旧可能な工法限定

### 4.2 操作方法（簡易マニュアル）

#### 照明操作

1. **LEDデスクライト**
   - 点灯: スイッチを押す
   - 調光: 明るさボタンで調整（段階的）
   - 消灯: スイッチ長押し

2. **間接照明（シーリング）**
   - 点灯: 壁スイッチON
   - 調色: 3000K（暖色）〜6500K（昼白色）
   - 調光: リモコンで明るさ調整

#### コンセント使用

1. **コンセントタップ**
   - 最大4口まで使用可能
   - 大型家電（電子レンジ等）は禁止
   - 負荷合計: 1500W以下

2. **延長コード**
   - 1段延長のみ
   - タコ足配線厳禁

#### デスク使用

1. **耐荷重**
   - デスク: 最大30kg
   - 収納棚: 最大20kg/段

2. **調整**
   - 水平行が傾いた場合、L型金具で再調整

### 4.3 メンテナンス方法

| メンテナンス項目 | 頻度 | 方法 |
|----------------|------|------|
| **換気** | 1時間ごとに30秒以上 | 扉を開けて空気を入れ替え |
| **照明清掃** | 月1回 | 柔らかい布で埃を拭き取る |
| **配線確認** | 月1回 | 被覆損傷・発熱チェック |
| **棚の緩み確認** | 3ヶ月に1回 | ネシの緩みを確認・締め直し |
| **壁紙のメンテナンス** | 汚れた際 | 柔らかい布で水拭き（強く擦らない） |

**注意点**:
- シンナー・ベンジン等の溶剤は使用禁止
- 過度な湿気は避ける
- 直射日光を当てない（壁紙の日焼け防止）
