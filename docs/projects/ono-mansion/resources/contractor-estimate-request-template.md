## 見積もり到着後のチェックリスト

業者から見積もりが到着したら、以下の項目を確認してください。

### 1. 基本情報の確認

- [ ] 業者名・担当者名
- [ ] 見積もり作成日
- [ ] 見積もり有効期限
- [ ] 見積もり番号

### 2. 工事内容の確認

- [ ] 工事対象（ユニットバス・給湯器・トイレ・洗面台）
- [ ] 工事範囲（撤去・設置・配管工事・養生）
- [ ] 使用資材（指定資材の反映有無）
- [ ] 工期（着手日・完了日）

### 3. 費用の確認

- [ ] 資材費（別途購入の場合は「なし」）
- [ ] 工事費（本体工事費・配管工事費別）
- [ ] 諸経費（養生費・廃材処分費・交通費等）
- [ ] 消費税（10%）
- [ ] 総額（税込）

### 4. 条件の確認

- [ ] 保証期間（通常1〜5年）
- [ ] 保証内容（不具合時の無償修理範囲）
- [ ] 追加費用の有無（現場調査後の追加見積もり等）
- [ ] キャンセル料（契約解除時の違約金）

### 5. その他の確認

- [ ] 施工実績の有無
- [ ] 担当者の資格・経験
- [ ] アフターサービスの内容
- [ ] 施工保険の加入有無

## 業者選定のポイント

### 1. 価格だけで選ばない

- 安さだけで選ぶと品質低下・トラブルのリスク
- 適正価格・納得のいく説明がある業者を選ぶ

### 2. 相見積もりを取得

- 必ず3社から見積もりを取得
- 価格・工事内容・対応を比較検討

### 3. 現場調査の実施

- 正確な見積もりのために現場調査が必須
- 図面のみの見積もりは追加費用のリスク

### 4. 見積もりの内訳を確認

- 一括費用ではなく、内訳が明記されているか
- 追加費用の可能性があるか確認

### 5. 保証・アフターサービス

- 保証期間が十分か（最低1年〜3年）
- アフターサービスの体制が整っているか

### 6. 担当者の信頼性

- 説明が分かりやすいか
- 質問に丁寧に答えるか
- スムーズなコミュニケーションが取れるか
