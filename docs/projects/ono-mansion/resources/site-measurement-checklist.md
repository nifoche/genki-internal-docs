# 現場測量チェックリスト

> 作成日: 2026-02-08
> 対象: 小野マンション401号室（ボタニカル）・402号室（猫多頭飼い）
> 目的: 既設寸法確認・資材発注前の最終確認

---

## 測量の目的

リフォーム工事をスムーズに進めるため、以下の項目について事前に詳細な寸法を確認します。

**確認事項**:
- 既設設備の寸法（交換用資材の適合確認）
- 設置スペースの確認（資材搬入経路）
- 配管・配線位置の確認（工事計画立案）
- 不具合箇所の特定（追加工事予測）

---

## 測量準備

### 必要道具

| 道具 | 用途 | 推奨 |
|------|------|------|
| メジャー（5m以上） | 一般的な寸法測定 | 金属性・巻き取り式 |
| メジャー（10m） | 大きな部屋の対角線測定 | 金属性・巻き取り式 |
| メモ帳・ペン | 寸法記録 | A5サイズ以上 |
| カメラ（スマホ） | 記録撮影 | スマホでOK |
| 懐中電灯 | 狭い場所・暗い場所の確認 | 小型・LED |
| 脚立 | 高い場所の測定 | 2m程度 |
| レーザー距離計 | 高精度測定（オプション） | あれば便利 |

### 測量時の注意点

- [ ] メジャーは水平・垂直に保つ
- [ ] 同じ箇所を2回測って確認
- [ ] 凸凹がある場合は最も狭い部分を測る
- [ ] 写真は寸法が分かるように撮る
- [ ] 不具合箇所は写真＋メモで記録

## 測量後の作業

### 1. データ整理

- [ ] 全寸法をエクセルや表計算ソフトに入力
- [ ] 各部屋の面積を計算
- [ ] 資材数量を再計算（ロス込み）
- [ ] 不具合箇所リスト作成

### 2. 資材発注準備

- [ ] 既設寸法と資材寸法の照合
- [ ] 設置可否の最終確認
- [ ] 第2候補商品の寸法確認
- [ ] 発注数量の最終決定

### 3. 業者への情報提供

- [ ] 測量結果を業者に送付
- [ ] 不具合箇所の写真共有
- [ ] 設置可否の確認依頼
- [ ] 見積もり依頼
