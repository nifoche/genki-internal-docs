## 401号室: 低投資ボタニカル（Week 2・5時間）

### ステップ1: 壁穴あけ（1時間）

**目的**: 簡易棚設置のための下穴加工

**手順**:
1. 簡易棚設置位置にマーキング
   - 高さ: 視線の高さ（約150cm）
   - 水平: 水準器で水平確認
   - 間隔: 2箇所（60cm間隔）

2. ドリルで壁穴あけ
   - ドリルビス径: 6mm
   - 深さ: 30mm
   - 角度: 壁に対して垂直

3. プラグの取り付け
   - プラグ種類: プラスチックアンカー
   - 打ち込み: ハンマーで水平に
   - 確認: プラグがしっかり固定されている

**確認項目**:
- [ ] 設置位置の水平確認
- [ ] 穴の深さ・角度適切
- [ ] プラグの正確な取り付け

**所要時間**: 1時間

### ステップ2: 簡易棚設置（1時間）

**目的**: 壁に簡易棚を固定

**手順**:
1. 壁に棚を固定（2箇所）
   - ネジ径: 4mm
   - 長さ: 40mm
   - トルク: 手で締めた後+1/4回転

2. 水準器で水平確認
   - 許容誤差: ±2mm以内
   - 調整: 必要に応じて微調整

3. 耐荷重テスト
   - 荷重: 5kg（観葉植物2〜3鉢分）
   - 時間: 1分間保持
   - 確認: 棚がたわまない

**確認項目**:
- [ ] ネジの締め付け確認
- [ ] 水平確認（±2mm以内）
- [ ] 耐荷重テスト（5kg・1分）

**所要時間**: 1時間

### ステップ3: LEDライト設置（2時間）

**目的**: 植物育成用照明と装飾用照明の設置

**手順**:
1. LED育成ライト設置（デスク横・窓際）
   - 位置: 植物の上30cm
   - 角度: 下向き30度
   - 固定: スタンドで安定

2. LEDテープライト設置（壁装飾用）
   - 貼付位置: 壁の上部周り
   - 貼付方法: 裏面粘着テープ
   - コーナー処理: 曲がりやすいように緩やかに

3. コンセント確認・延長コード準備
   - コンセント位置: 照明近く
   - 延長コード長: 2m
   - 安全確認: コードが床に這わない

4. タイマー設定
   - ON時間: 7時（朝）
   - OFF時間: 19時（夜）
   - 週間: 毎日同じ時間

**確認項目**:
- [ ] LED育成ライトの角度・位置適切
- [ ] テープライトの貼付強度確認
- [ ] 延長コードの安全確認
- [ ] タイマーの動作確認

**所要時間**: 2時間

### ステップ4: 観葉植物配置（1時間）

**目的**: 日当たりを考慮して植物を配置

**手順**:
1. 日当たり確認
   - 方角: 南向きの窓
   - 日照時間: 4-6時間/日
   - 距離: 窓から1-2m

2. 植物配置
   - ポトス: 窓際（耐陰性あり）
   - ドラセナ: 明るい場所
   - サンスベリア: 半日陰
   - ユッカ: 日当たり良好
   - 多肉植物: 直射日光OK

3. 水やりスケジュール作成
   - 頻度: 週1回（日曜日）
   - 時間: 午前中
   - 量: 土の表面が乾いてから

4. 肥料散布スケジュール作成
   - 頻度: 月1回（第1日曜日）
   - 種類: 固形肥料・液体肥料
   - 時期: 春〜秋

**確認項目**:
- [ ] 日当たり条件の確認
- [ ] 植物の配置適切
- [ ] 水やりスケジュール作成
- [ ] 肥料散布スケジュール作成

**所要時間**: 1時間

## 402号室: 水回りリフォーム（Week 5-6・約2週間）

### ステップ1: 既存設備撤去（3日間）

**目的**: 既存の水回り設備を撤去

**手順**:
- デイ1: ユニットバス撤去
  - 給水・排水止め
  - 壁・床解体
  - 搬出・処分

- デイ2: 給湯器撤去
  - ガス/電源止め
  - 配管取り外し
  - 搬出・処分

- デイ3: トイレ・洗面台撤去
  - 給水・排水止め
  - 取り外し
  - 廃材処分

**確認項目**:
- [ ] 給水・排水完全停止
- [ ] 廃材適切処理
- [ ] 作業場所の清掃

**所要時間**: 3日間

### ステップ2: 配管工事（2日間）

**目的**: 新設備用の配管設置

**手順**:
- デイ4: 水回り配管確認
  - 既存配管の確認
  - 新設備との接続点確認
  - 必要に応じて配管変更

- デイ5: 追加配管工事
  - 追加配管設置
  - 接続テスト
  - 水圧確認

**確認項目**:
- [ ] 配管の適切な接続
- [ ] 水圧適切（1.5-2.5kg/cm²）
- [ ] 漏れなし

**所要時間**: 2日間

### ステップ3: ユニットバス設置（3日間）

**目的**: 新しいユニットバスを設置

**手順**:
- デイ6: ユニットバス搬入
  - 搬入経路確保
  - 部材確認
  - 仮置き

- デイ7: 設置・接続
  - 据付
  - 給水・排水接続
  - 電気配線接続

- デイ8: 動作確認
  - 給水テスト
  - 排水テスト
  - 温度調整テスト

**確認項目**:
- [ ] 給水正常（水圧適切）
- [ ] 排水正常（詰まりなし）
- [ ] 温度調整正常（40-60℃）
- [ ] 漏れなし

**所要時間**: 3日間

### ステップ4-6: 給湯器・トイレ・洗面台設置（各1日）

**目的**: 各設備の設置と動作確認

**手順**:
- 給湯器設置（デイ9）
  - 搬入・設置
  - 配管接続
  - 点火・温度確認

- トイレ設置（デイ10）
  - 搬入・設置
  - 給水接続
  - 洗浄動作確認

- 洗面台設置（デイ11）
  - 搬入・設置
  - 給水・排水接続
  - 排水確認

**確認項目**:
- [ ] 給湯器: 点火・温度正常
- [ ] トイレ: 洗浄・止水正常
- [ ] 洗面台: 給水・排水正常

**所要時間**: 各1日（計3日）

### ステップ7: 最終検査・引き渡し（1日）

**目的**: 全設備の最終確認

**手順**:
- 全設備動作確認
  - 給水: 全箇所
  - 排水: 全箇所
  - 温度: 給湯器

- 漏れチェック
  - 配管接続箇所
  - 設備接続箇所

- 清掃・引き渡し
  - 作業場所清掃
  - 説明書渡し
  - 操作説明

**確認項目**:
- [ ] 全設備動作正常
- [ ] 漏れなし
- [ ] 清掃完了
- [ ] 説明書・操作説明完了

**所要時間**: 1日

## 402号室: 猫多頭飼い設備設置（Week 7・1週間）

### ステップ1: クッションフロア敷設（3日間）

**目的**: 防水・滑り止めフロアの敷設

**手順**:
- デイ1: 既存フロア撤去
  - 既存フロア剥がし
  - 下地清掃
  - 廃材処分

- デイ2: 防水処理
  - 下地処理
  - 防水材塗布
  - 乾燥待ち

- デイ3: クッションフロア敷設
  - フロア搬入
  - 敷設（部屋の隅から）
  - スライド通り調整
  - 隙間確認

**確認項目**:
- [ ] 下地の平らさ
- [ ] 防水処理完了
- [ ] フロアの隙間なく敷設
- [ ] スライド通り適切

**所要時間**: 3日間

### ステップ2: 耐傷壁パネル設置（2日間）

**目的**: 猫の爪による壁傷防止

**手順**:
- デイ4: 壁面下地処理
  - 既存壁の状態確認
  - 下地処理
  - 平らさ確認

- デイ5: 壁パネル設置
  - パネル搬入
  - 設置（1m高さ）
  - コーキング処理
  - 高さ均一確認

**確認項目**:
- [ ] 下地処理完了
- [ ] パネルの1m高さ均一
- [ ] コーキング完了
- [ ] 取り付け強度確認

**所要時間**: 2日間

### ステップ3: 空気清浄機設置（1日）

**目的**: ペット臭の低減

**手順**:
- 空気清浄機搬入
- 設置場所決定
  - 場所: 部屋の中央
  - 高さ: 床から30cm以上
  - 周囲: 物から30cm以上空ける
- 動作確認
  - 電源ON
  - 風量確認
  - フィルター確認

**確認項目**:
- [ ] 設置場所適切
- [ ] 動作正常
- [ ] 風量十分

**所要時間**: 1日

### ステップ4: 24時間換気システム設置（1日）

**目的**: 常時換気で臭い蓄積防止

**手順**:
- 換気扇設置
  - 設置場所: 壁上部
  - ダクト設置
  - 電源接続
- タイマー設定
  - 連続運転: 24時間
  - 風量: 中程度
- 動作確認
  - 稼働音確認
  - 換気量確認

**確認項目**:
- [ ] 設置場所適切
- [ ] タイマー設定確認
- [ ] 24時間稼働確認
- [ ] 稼働音適切（40dB以下）

**所要時間**: 1日
