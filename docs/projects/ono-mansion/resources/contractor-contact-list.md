## 連絡先の使い方

## メモ

**業者選定のポイント**:
- [ ] 見積もりは3社から取得
- [ ] 現場調査を実施してもらう
- [ ] 保証期間・アフターサービスを確認
- [ ] 施工実績・口コミを確認

**契約前の最終確認**:
- [ ] 工事内容・工期・金額を最終確認
- [ ] 追加費用の有無を確認
- [ ] キャンセル料・解約条項を確認
- [ ] 契約書の内容を良く読む
