## 概要

本購入計画は、リフォーム資材の発注タイミング・手順・優先順位を具体化したものです。工事の進捗に合わせて資材を購入し、在庫保管スペースを最小限に抑えながらスムーズな施工を実現します。

**購入原則**:
- 工事開始の2週間前に発注
- 長納期資材（ユニットバス）は最優先発注
- DIY資材は施工週の週末に購入
- 大型資材は現場直送

- [ ] 納期が工事スケジュールに合うか
- [ ] 送料・代引き手数料確認
- [ ] 第2候補商品の把握

**大型資材（ユニットバス・給湯器）**:
- [ ] 設置条件確認（既設寸法・配管位置）
- [ ] 工事費別途見積もり取得
- [ ] 専門業者手配完了

**DIY資材（クロス・ドア等）**:
- [ ] 施工方法確認（YouTube等）
- [ ] 必要工具リスト作成
- [ ] 返品・交換ポリシー確認

### 購入手順

1. **URLから商品ページを開く**
2. **在庫状況・納期を再確認**
3. **カートに入れて支払い完了**
4. **注文確認メールを受け取り**
5. **納期をカレンダーに記録**
6. **到着予定日を施工スケジュールに反映**

## リスク管理

### 在庫切れ対策

- [ ] 第2候補商品URLをブックマーク
- [ ] 複数店舗で在庫比較
- [ ] メール在庫通知機能を設定

### 納期遅延対策

- [ ] 余裕を持った発注（2週間前）
- [ ] 施工スケジュールの調整可能日を確保
- [ ] 業者との連絡体制確立

### 品質問題対策

- [ ] 購入者レビュー確認（3.5星以上）
- [ ] 到着時の開封検品
- [ ] 不良品は即時返品手続き
