# DIY工具購入リスト

> 作成日: 2026-02-08
> 対象: 401号室（ボタニカル）・402号室（猫多頭飼い）
> 目的: DIY施工に必要な工具一式

---

## 概要

本リストは、リフォームのDIY施工部分（内装・建具・専用設備設置）に必要な工具をまとめたものです。

**対象作業**:
- クロス張替え（90m²）
- ドア交換（6枚）
- クッションフロア施工（45m²）
- 観葉植物配置・LED照明設置
- ウォールシェルフ取付け
- 空気清浄機設置
- 24時間換気設置
- 耐傷壁パネル施工

## 工具メンテナンス

### 使用前の準備

- [ ] 工具の動作確認（電動工具）
- [ ] 刃先の確認（カッター・ドリル）
- [ ] 充電（電動ドリル）

### 使用中の注意点

- [ ] 電動ドリル: 連続使用で過熱→休憩
- [ ] カッター: 替刃はこまめに交換
- [ ] 水準器: 破損に注意

### 使用後のメンテナンス

- [ ] 刃先の清掃・油差し
- [ ] 電動工具の充電
- [ ] 破損工具の交換
- [ ] 保管場所の整理
