## 評価軸と定義

### 1. 需要予測（30%加重）

市場規模とターゲット層の広さを評価

| スコア | 定義 |
|--------|------|
| 10点 | 非常に広いターゲット層（市場規模大） |
| 8-9点 | 広いターゲット層（市場規模中〜大） |
| 6-7点 | やや狭いターゲット層（市場規模小〜中） |
| 4-5点 | 狭いターゲット層（市場規模小） |
| 1-3点 | 非常に狭いターゲット層（市場規模極小） |

### 2. 初期投資（20%加重）

投資額の少なさを評価

| スコア | 定義 |
|--------|------|
| 10点 | 10万円未満 |
| 8-9点 | 10〜50万円 |
| 6-7点 | 50〜100万円 |
| 4-5点 | 100〜200万円 |
| 1-3点 | 200万円以上 |

### 3. 回収期間（25%加重）

回収の速さを評価

| スコア | 定義 |
|--------|------|
| 10点 | 1年未満 |
| 8-9点 | 1〜2年 |
| 6-7点 | 2〜3年 |
| 4-5点 | 3〜4年 |
| 1-3点 | 4年以上 |

### 4. 競合差別化（15%加重）

他物件との差別化の強さを評価

| スコア | 定義 |
|--------|------|
| 10点 | 競合が皆無（完全にユニーク） |
| 8-9点 | 競合が少なく、強い差別化がある |
| 6-7点 | 競合がいるが、ある程度の差別化がある |
| 4-5点 | 競合が多く、差別化が弱い |
| 1-3点 | 競合が多く、差別化がほとんどない |

### 5. リスク（10%加重）

失敗の可能性の低さを評価

| スコア | 定義 |
|--------|------|
| 10点 | リスク極小（成功確率90%以上） |
| 8-9点 | リスク小（成功確率70〜90%） |
| 6-7点 | リスク中（成功確率50〜70%） |
| 4-5点 | リスク大（成功確率30〜50%） |
| 1-3点 | リスク極大（成功確率30%未満） |

## 加重スコア計算

加重スコア = Σ（評価点 × 重み）

| コンセプト | 需要予測<br>(30%) | 初期投資<br>(20%) | 回収期間<br>(25%) | 競合差別化<br>(15%) | リスク<br>(10%) | **加重総合点** | **順位** |
|-----------|-----------------|-----------------|-----------------|-------------------|-------------|--------------|--------|
| **A: テレワーク** | 8×0.3=2.4 | 4×0.2=0.8 | 6×0.25=1.5 | 9×0.15=1.35 | 6×0.1=0.6 | **6.65** | **3位** |
| **B: クリエイター** | 5×0.3=1.5 | 3×0.2=0.6 | 5×0.25=1.25 | 10×0.15=1.5 | 4×0.1=0.4 | **5.25** | **5位** |
| **C: シンプル** | 9×0.3=2.7 | 6×0.2=1.2 | 7×0.25=1.75 | 4×0.15=0.6 | 7×0.1=0.7 | **6.95** | **4位** |
| **D: ボタニカル** | 8×0.3=2.4 | 10×0.2=2.0 | 10×0.25=2.5 | 6×0.15=0.9 | 9×0.1=0.9 | **8.70** | **1位** |
| **E: 多頭飼い** | 7×0.3=2.1 | 7×0.2=1.4 | 9×0.25=2.25 | 8×0.15=1.2 | 6×0.1=0.6 | **7.55** | **2位** |

## 市場調査データに基づく補足評価

### 競合物件賃料分析

**現在の小野マンション**: 5.5万円（1LDK、43.91m²、築50年）

**競合物件**:
- 1K物件平均: 6〜6.8万円
- ペット可1K物件: 6.5〜7.5万円（**+5,000〜+10,000円のプレミアム**）
- 2K物件（築40年以上）平均: 4.03万円

**評価**:
- **D: ボタニカル（4.3〜4.5万円）**: 競合物件より安価で差別化可能、入居率向上が期待
- **E: 多頭飼い（5.5〜6.5万円）**: ペット可プレミアムを考慮すると妥当、ただし競合が存在

### ペット可物件の実態調査

**調査結果**:
- ペット可物件は賃料が**+5,000〜+10,000円高い**
- 敷金が**1ヶ月分追加**（約6.5万〜13.9万円）
- 多頭飼い可物件は**非常に少ない**
- 小型犬・猫2匹までが制限

**評価**:
- **E: 多頭飼い**はニッチ市場で競合が少ない（**差別化強**）
- ただし、近隣トラブルのリスクがある（**リスク中**）

## 推奨アクション

### Phase 1: 市場実験（2週間）

**推奨コンセプト**: **D: ボタニカル特化**

**理由**:
1. 加重総合点が最高（8.70）
2. 初期投資が極めて低い（3.4万円）
3. 回収期間が最短（7ヶ月）
4. リスクが極めて低い（失敗しても影響少）

**成功基準**:
- 問い合わせ5件以上
- 内見2件以上
- 成約1件以上

**成功した場合**: Phase 2（本格展開）へ進む
**失敗した場合**: E: 多頭飼いに切り替え

### Phase 2: 本格展開（1ヶ月）

**推奨コンセプト**: **E: 多頭飼い専用** または **A: テレワーカー専用**

**理由**:
- E: 多頭飼いは加重総合点2位（7.55）、回収1年
- A: テレワークは需要予測8点・差別化9点

**条件**:
- ヒアリングで需要が確認された場合
- 賃料プレミアム（+1〜1.5万円）の受容性が確認された場合
- 投資余力がある場合（50〜300万円）
