## プロジェクト基本情報

### 目的
自社物件のバリューアップ実証（2部屋）と、その後のオーナー向けリフォームパッケージの商品化実験

### ターゲット層
**「綺麗」を超えた「唯一無二」の価値（猫多頭飼い、特定趣味等）を求める層**
- 偏愛ニーズを攻める（2物件改装）
- 従来のリフォーム競争とは別次元の価値提案

### 成功指標（KPI）

| 指標 | 目標値 |
|------|--------|
| **賃料アップ率** | 周辺相場比 +20% 以上の成約 |
| **成約スピード** | 工事完了から2ヶ月以内の入居申し込み |
| **投資回収** | リフォーム費用の回収シミュレーションの確立 |

### チーム体制・役割

| 役割 | 担当内容 |
|------|----------|
| **プロジェクトマネジメント** | 全体進捗管理、収支・予算管理、パッケージ設計 |
| **企画** | 市場調査（ニッチニーズ）、コンセプト立案、仕様・設備選定、マーケティング素材作成 |
| **施工・運用** | 施工業者選定・管理、工程管理、管理会社・リーシング調整 |

### リサーチ戦略
**SUUMOを活用した「逆算型」市場分析**

日本最大の不動産ポータル「SUUMO」「HOME'S」を使い、特定のこだわり条件（例：猫3匹以上、防音、大型バイク可など）での「検索ヒット数」と「周辺の競合物件数」を調査。市場の供給不足を数値化し、そこから逆算して「確実に勝てる（家賃を上げても選ばれる）コンセプト」を導き出す。

### 納期（募集開始）
- **物件1・物件2**: GrowthHackまで（6月初旬）には募集開始
- 入居も決まると望ましい（3月末理想、4月末最終）

## プランC概要

### 採用コンセプト

**401号室**: 低投資ボタニカル（3.4万円 + 水回り167.5万円）
- 水回りリフォーム（ユニットバス・給湯器・トイレ・洗面台）
- 観葉植物5鉢（ポトス・ドラセナ・サンスベリア・ユッカ・多肉植物）
- LED育成ライト2台・LEDテープライト5m
- 簡易棚2個
- 想定賃料: 5.8万円（+0.3万円）
- 回収期間: 11ヶ月（コンセプト分のみ） / 約2.8年（総投資額）

**402号室**: 猫多頭飼い（28万円 + 水回り167.5万円）
- 水回りリフォーム（ユニットバス・給湯器・トイレ・洗面台）
- クッションフロア敷設（防水・滑り止め）
- 耐傷壁パネル設置（1m高さ）
- 空気清浄機・24時間換気システム
- 想定賃料: 6.15万円（+0.65万円）
- 回収期間: 28ヶ月（2.3年）

### 投資額内訳

| 項目 | 401号室 | 402号室 | 合計 |
|------|---------|---------|------|
| **通常リフォーム** | 167.5万円 | 167.5万円 | **335万円** |
| **コンセプト追加費用** | 3.4万円 | 28万円 | **31.4万円** |
| **総投資額** | **170.9万円** | **195.5万円** | **366.4万円** |

### 月間キャッシュフロー

```
基礎賃料: 11万円（5.5万円 × 2）
上乗せ賃料: +0.95万円
合計: 11.95万円/月

年間収益: 143.4万円
2年間収益: 22.8万円（コンセプト投資回収率72.6%）
2.4年間収益: 27.55万円（コンセプト投資回収率87.7%）
```

## リスク管理

### 401号室特有のリスク

**植物が枯れるリスク**:
- 確率: 20%
- 影響: コンセプト破綻・問い合わせ減少
- 対策:
  - 水やりのカレンダー作成（週1回）
  - LEDライトタイマー設定（毎朝7時ON・夜7時OFF）
  - 週1回の状態確認（日曜日に実施）
  - 予備の観葉植物を確保

**内装傷みリスク**:
- 確率: 30%
- 影響: 退去時の原状回復費用
- 対策:
  - 壁紙の選定（汚れに強いタイプ）
  - 定期的な清掃（月1回）
  - 入居者へのマニュアル提供

### 402号室特有のリスク

**ペットの臭いリスク**:
- 確率: 40%
- 影響: 近隣トラブル・苦情
- 対策:
  - 24時間換気システム設置
  - 空気清浄機常時稼働
  - 消臭剤使用
  - 近隣への事前説明

**空室リスク**:
- 確率: 50%（猫多頭飼いは入居者決定まで時間がかかる）
- 影響: 収益減
- 対策:
  - SUUMO・HOMES掲載強化
  - ペット専用SNSアカウント運用
  - ペットショップとの提携
  - 紹介制度（1ヶ月無料）

## 検討事項・メモ

### コンセプトに関する検討事項

**猫コンセプト**:
- 猫は有料候補だが、調査や検証が必要
- 猫で進める場合は、どうせ汚れるので、スケルトンに近いのが良い
- ペットの臭いリスク：24時間換気システム設置、空気清浄機常時稼働で対応

**ダイナミックな選択肢**:
- 2部屋を1部屋にするダイナミック策もあり得る（費用はかかる）
- ニコイチにすると、水回りを1つにできるメリットもある
- 1物件はお金をかけずに無骨なものが良い

**将来的な可能性**:
- 民泊も可能性として検討（将来的にはあり得るが、今回はない）
